M3U Format Explained

An M3U playlist is a simple text file that contains a list of media streams. Here's how the format works:

Header

#EXTM3U

Every M3U playlist starts with this header to identify the file format.

Entry Format

Each channel/stream has two lines:

#EXTINF:-1 tvg-id="..." tvg-name="..." tvg-logo="..." group-title="...",Display Name
https://stream-url.m3u8

Attributes Breakdown

#EXTINF:-1 Extended info tag. -1 means live stream (no duration).
tvg-id Unique identifier for the channel
tvg-name Channel name (used for matching)
tvg-logo URL to the channel's logo image
group-title Category name for grouping channels
,Display Name Text after the comma is the display name shown in the app
URL (next line) The actual stream URL (HLS .m3u8, MP4, etc.)

How to Test

  1. Open ShallowTV on your device
  2. On first launch, paste the URL above when prompted
  3. Or go to Library → tap + → Add from URL
  4. Paste: https://shallowtv.com/sample.m3u
  5. The test channels will appear in the Home tab
