About This Playlist

This sample playlist contains official Apple HLS test streams provided by Apple for developer testing. These streams are safe, legal, and commonly used for testing media player applications.

Included Test Streams

  • Bipbop Classic (4x3) - Basic HLS test stream
  • Bipbop Widescreen (16x9) - Widescreen HLS test stream
  • Advanced HLS (FMP4) - Fragmented MP4 variant
  • Dolby Vision + Atmos - HDR and spatial audio test
  • Historic Planet (3D Video) - 3D video for Vision Pro

How to Test

  1. Open ShallowTV on your device
  2. On first launch, paste the URL above when prompted
  3. Or go to Library → tap + → Add from URL
  4. Paste: https://shallowtv.com/sample.m3u
  5. The test channels will appear in the Home tab

Source

All streams are sourced from Apple's official developer streaming examples:
